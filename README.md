# PROYECTO
Proyecto Intermodular de Administración de Sistemas Informáticos en Red

Este proyecto simula el funcionamiento de VirusTotal mediante el uso
de herramientas OSINT para analizar URLs y detectar posibles amenazas.

## Tecnologías
- PHP
- MySQL
- OSINT
- HTML/CSS

## Funcionalidades
- Análisis de URLs
- Almacenamiento de resultados
- Historial de análisis
- Sistema de usuarios
